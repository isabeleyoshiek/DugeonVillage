# DugeonVillage
É um jogo em que busco trazer um ar divertido e interativo entre personagens como fadas , NPC´S e entre outros entre o jogador e jogo aqui apresentado. O jogo é composto de laços de repetição, condicionais e comandos interativos com quem se propõem a jogar
# Dungeon Village

Projeto desenvolvido em Linguagem C.

## Funcionalidades

- Menu principal
- Tutorial
- Escolha de armas
- Movimentação na vila
- Sistema de chave e porta
- Primeira masmorra
